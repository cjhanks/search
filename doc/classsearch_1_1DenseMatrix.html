template<typename Type_, Order order_ = Order::DEFAULT>
class search::DenseMatrix< Type_, order_ >

Template Parameters
Type_Underlying type to be stored in this class.
order_This class can store in Row major OR Col major.

A dense matrix is a 2D matrix which can be accessed by row/col.
