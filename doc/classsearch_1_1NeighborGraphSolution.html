Detailed Description

template<typename NodeType_, typename MatrixType_>
class search::NeighborGraphSolution< NodeType_, MatrixType_ >

Template Parameters
NodeType_What data type is being stored in this solution.
MatrixType_What is the underlying matrix storing data.

This class holds a mapped solution for shortest path. It stores internally the Node mapping to the matrix index.
