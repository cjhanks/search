Detailed Description

template<typename NodeType_, typename EdgeType_>
class search::NeighborGraph< NodeType_, EdgeType_ >

Template Parameters
NodeType_What data type is being stored in this solution.
EdgeType_What data type is being stored in the edge.

The NeighborGraph is used to store a neighbor graph in graph form, opposed to the NeighborGraphSolution which stores it in matrix form.
